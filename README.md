# COS 106 - Student Portfolio Project

## Overview
This is a 5-page personal portfolio website built as a term project for COS 106 (Introduction to Computing) at MIVA Open University.

## Student Information
- **Name:** Oluwadara Bolaji
- **Level:** 100 Level
- **Course:** Software Engineering
- **University:** MIVA Open University

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

No frameworks, libraries, or build tools were used in this project.

## Pages
1. **Homepage** (`index.html`) - Welcome page with hero section, quick facts, and course table
2. **About Me** (`pages/about.html`) - Education background, skills, hobbies, and introduction video
3. **Projects** (`pages/projects.html`) - Showcase of 3 projects with descriptions and links
4. **Academic Planner** (`pages/planner.html`) - Task management tool for tracking assignments
5. **Contact** (`pages/contact.html`) - Contact information and message form

## Features
- Responsive design (works on desktop, tablet, and mobile)
- Mobile hamburger navigation menu
- CSS Grid and Flexbox layouts
- CSS animations and transitions
- HTML5 multimedia elements (video)
- Interactive task planner
- Contact form with validation

## Colour Scheme
- **Navy:** `#1a3a5c` (primary)
- **Gold:** `#f0c040` (accent)
